Bienvenue sur FlexiTime !

 

FlexiTime est un outil permettant de gérer des emplois du temps. Il a été développé pour une utlisation dans un cadre scolaire tel une école d'ingénieurs ou une université.

FlexiTime se compose de deux modes de fonctionnement principaux :
- Le mode gestion des ressources permet d'ajouter, modifier ou encore supprimer une ressource.
On entend par ressource : une salle, un professeur, un matériel, etc...
- Le mode exploitation permet de créer des emplois du temps à partir des ressources; ils sont également modifiables à souhait.

Pour plus de renseignement sur l'utilisation de ce logiciel, rendez-vous dans l'aide en ligne accessible par le menu Aide puis Aide en ligne...

Ce logiciel a été réalisé dans le cadre d'un projet de Génie Logiciel pour l'école Ingénieurs2000.
FlexiTime est entièrement codé en Java. Bien sûr il n'est pas exempt de bugs !
